Welcome To My Calculator Webpage!

What is the Purpose of this Webpage?

This webpage is connected to a behind-the-scenes Calculator that accepts inputs via an HTML form. You can find this form at the equivalently named link on the Navbar. This Calculator will process your inputs, and then the webpage will display the results for you to see! These results are also then stored in a table along with the inputs you gave to achieve that result. In fact, if you click on "Calculation History" on the Navbar, you'll see all the results that have been calculated in a pretty handy table! This table is even searchable and groupable by the operation column, should that be something you find the need to do.

Along with the calculator and its history, there are some reference articles included on this page as well. Links to these articles can also be found in the Navbar. These articles have a plethora of information about working with and understanding Python, and by extension other Object-Oriented Programming languages. These articles even use examples from the underlying code of our calculator to help illustrate the main points they are trying to get across. Check them out!
